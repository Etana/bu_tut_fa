Bonjour à toutes et à tous,

Pour célébrer Pâques comme il se doit ForumActif vous propose une série d'animations sous forme de codes, simples à installer pour habiller votre forum


________________________________________________________________

Pour installer ces codes, rien de plus simple :

Panneau d'administration Modules Gestion des codes Javascript Créer un nouveau javascript


1. Indiquez un titre de votre choix au script (ex: Noël)
2. Cochez les cases correspondantes aux endroits où vous souhaitez voir le script (ex: Toutes les pages)
3. Collez le code de votre choix parmi ceux qui vous sont proposés


___________________________________________________________________________
