Nouvelle fonctionnalité: Le tag des utilisateurs @


Bonjour à tous,

Nous sommes heureux de vous annoncer qu'une nouvelle fonctionnalité est disponible pour vos forums : le tag des utilisateurs dans les posts.
Cette nouvelle option permet de mentionner un utilisateur à laide du symbole @ afin de le notifier dans un post.

Pour activer cette option, rendez-vous dans le panneau dadministration:

PA > Général > Messages et Emails > Configuration > Messages > Autoriser le tag des utilisateurs dans les messages: Non / Groupes / Tous les membres.



Il est possible d'activer l'option pour certains groupes uniquement:



Si l'on souhaite mentionner un utilisateur, il suffit d'ajouter un @ avant son pseudo.
Exemple: @Vivi

Si le pseudo comporte un espace, il faut alors mettre le pseudo entre guillemets.
Exemple: @The Godfather

Code:
@"The Godfather"

Les pseudos contenant des caractères spéciaux ne sont pas gérés pour la mention

De plus, la notification des utilisateurs, respectent les conditions de permissions établis sur le forum. Ainsi si un utilisateur est mentionné dans un sujet auquel il n'a pas accès, il ne recevra pas la notification correspondante.

Lorsque le pseudo de l'utilisateur est mentionné, ce tag peut être affiché dans le centre de notifications et dans la toolbar (les notifications par email ne sont pas disponibles pour cette option):



Pour cela, il convient de cocher la case correspondante dans le panneau de notifications, Profil > Notifications:



À noter: Les pseudos dans la balise citation (quote) sont désormais précédés d'un @ cliquable, mais qui n'entraine pas l'envoi d'une notification:



En espérant que cette nouveauté pour vos forums vous plaisent!

Cordialement,

L'équipe ForumActif

Nouvelle fonctionnalité: Le tag des mots-clés #


Bonjour à tous,

Nous avons le plaisir de vous annoncer une nouvelle fonctionnalité pour vos forums Forumactif : le tag de mots-clés dans les posts.

Cette nouvelle option permet de taguer un mot-clé à laide du symbole # afin de suivre des mots-clés spécifiques ou dafficher la liste des messages dans lesquels ils sont cités.

Pour activer cette option, rendez-vous dans le panneau dadministration:

PA > Général > Messages et Emails > Configuration > Messages > Autoriser le tag des mots-clés dans les messages: Non / Groupes / Tous les membres.




Il est possible d'activer la fonctionnalité seulement pour des groupes choisis:



Sur le forum:

Il est possible de suivre une liste de mots tagués définis dans Profil > Tags :



Lorsquun mot-clé suivi est tagué, une notification est affichée dans le centre de notifications et dans la toolbar:



Les notifications sont reçues seulement pour les mots-clés spécifiés comme suivis.

De plus, dans les messages, les mots précédés d'un # sont cliquables et renvoient à la liste des messages où le mot-clé apparaît.

Les mots-clés contenant des caractères spéciaux ne sont pas gérés lors du tag


Enfin, il est désormais possible dans la recherche (Rechercher, Recherche Avancée, Widget Recherche) de procéder par mot-clé:





À vos tags!

L'intérêt pour les hashtags est de faciliter les recherches et de pouvoir suivre un mot-clé en particulier.
Par exemple si vous vous abonnez au hashtag #invision, vous allez recevoir une notification dès que quelqu'un utilise le hashtag dans l'un de ses messages.

Concernant le tag utilisateur, il permet de vous envoyer une notif dès que quelqu'un utilise @votrepseudo à l'intérieur de son message. Et donc d'attirer votre attention sur le message en question.
Comme cette notif est sur oui par défaut (toolbar) si vous n'avez pas encore désactivé cette notif, vous allez recevoir une notif dans la toolbar à votre prochaîne connexion.

Pour l'impact sur nos forums il sera nul tant que personne n'utilisera les @ ou #

Question bête, sans doute:
Mais où se trouve le "#" sur le clavier?

Alt+035

Si tu as des membres qui ont des difficultés avec les raccourcis clavier, tu peux installer ce js dans la gestion des javascripts en cochant sur toutes les pages. Ce js installe un bouton sur les deux éditeurs juste après le bouton citation.

Code:
$(function(){$(function(){
 
 $('<a class="sceditor-button" unselectable="on" title="Hastag"><div unselectable="on" style="background-image:url(http://i55.servimg.com/u/f55/11/93/85/24/hastag10.gif)">Hastag</div></a>').insertAfter(' .sceditor-button-quote ').click(function(){
    $('#text_editor_textarea').sceditor("instance").insertText(" #");
  });
 
})});
