Bonjour à toutes et tous,

Le printemps est installé et la course aux œufs va pouvoir commencer !

A cette occasion, ForumActif vous propose une série d'animations sous forme de codes, simples à installer pour habiller votre forum



________________________________________________________________

Pour installer ces codes, deux méthodes s'offrent à vous :


La première, il vous suffit d'insérer le code de votre choix dans :
Panneau d'administration Général Forum | Configuration Description du site
Voir un aperçu:

Résultat : Votre décoration s'affichera sur toutes les pages de votre forum.



La deuxième, vous pouvez ajouter plusieurs codes à la fois via les balises :
Panneau d'administration Général Promotion du forum | Moteurs de recherche Balises supplémentaires
Voir un aperçu:




Résultat : L'objet de décoration s'affichera sur le portail, la page d'accueil, les pages des sous forums et les pages des sujets sur votre forum.
