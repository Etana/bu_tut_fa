Décorez vos forums pour les fêtes de fin d'année !

 par Vivi le Ven 21 Nov 2014 - 9:59

Bonjour à toutes et tous,


A l'occasion de Noël et la fin d'année, ForumActif vous propose une série d'animations sous forme de codes, simples à installer pour habiller votre forum aux couleurs de Noël

Attention, tous les scripts ne doivent pas être installés au même endroit. Merci de bien regarder les différents chemins qui vous sont fournis.
Le sujet peut également être long à charger, merci de votre patience.



Pour installer ces 2 codes:

Panneau d'administration Affichage Page d'accueil Généralités
Décoration sur la page d'accueil



  • Animation musicale et visuelle (avec bouton ON/OFF)
    (un peu long à charger les premières fois)
Code:
<embed pluginspage="http://www.macromedia.com/go/getflashplayer" src="http://sd-1.archive-host.com/membres/up/132720629421430912/FDF/Decoration/Noel/noel2.swf" width="100%" height="650" type="application/x-shockwave-flash" wmode="transparent" quality="high" allowScriptAccess="always"></embed>


  • Image animée "Joyeux Noël" traduit dans différentes langues
Code:
<embed pluginspage="http://www.macromedia.com/go/getflashplayer" src="http://sd-1.archive-host.com/membres/up/132720629421430912/FDF/Decoration/Noel/ban1.swf" width="450" height="450" type="application/x-shockwave-flash" wmode="transparent" quality="high" allowScriptAccess="always"></embed>


Pour installer ces codes:


Panneau d'administration Modules Gestion des codes Javascript Créer un nouveau javascript

1. Indiquez un titre de votre choix au script (ex: Noël)
2. Cochez les cases correspondantes aux endroits où vous souhaitez voir le script (ex: Toutes les pages)
3. Collez le code de votre choix parmi ceux qui vous sont proposés
