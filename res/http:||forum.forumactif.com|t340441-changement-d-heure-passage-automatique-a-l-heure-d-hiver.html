Changement d'horaire
sur Forumactif

Bonjour,

Dans la nuit du samedi 27 octobre au dimanche 28 octobre nous avons changé d'heure. À 3h du matin, nous avons réglé nos horloges à 2h.

Les forums Forumactif sont dotés dune technologie leur permettant de détecter et gérer automatiquement ce changement que ça soit pour les admins comme les membres. Vous n'avez donc eu aucune manipulation à effectuer pour bénéficier du changement d'heure !

Pour rappel, vous pouvez gérer le fuseau horaire d'un forum :
  • pour les administrateurs : PA → Général → Forum - Configuration → Fuseau horaire
  • pour les membres : Profil → Préférences → Fuseau horaire
