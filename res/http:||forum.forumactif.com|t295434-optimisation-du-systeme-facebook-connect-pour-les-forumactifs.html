Optimisation du système Facebook Connect pour les forumactifs.

 par The Godfather le Mar 18 Jan 2011 - 17:12

Chers utilisateurs Forumactif,

Dans la continuité de nos efforts pour vous offrir des forums intuitifs, sécurisés et à la pointe de la technologie, nous avons effectué une optimisation de loption « Facebook Connect ». Cela, dans le but daméliorer le fonctionnement et la sécurité de vos forums.

Dorénavant, un nouveau champ en relation avec cette application et sintitulant « Clé secrète » a fait son apparition dans votre panneau dadministration >> Modules >> Facebook Connect

Si vous êtes administrateur du forum, 2 cas se présentent :

  1. Vous avez déjà activé le système « Facebook Connect » sur votre forum : Dans ce cas, pour récupérer votre Clé secrète, il vous suffit de vous connectez sur le compte facebook avec lequel vous avez créé lapplication. Puis de vous rendre sur la page "Mes applications" et copier/coller la Clé secrète correspondante à l'application facebook connect que vous avez créée pour votre forum.



  2. Vous n'avez jamais activé le système « Facebook Connect » sur votre forum : dans ce cas, c'est très simple. La Clé secrète est fournie directement après la création de l'application.




Notez bien qu'afin de pouvoir profiter de cette optimisation, les admins ayant modifié le template index_body doivent prendre en compte le changement ci-dessous :

Il faut remplacer le code:

Code:
onlogin="facebook_login()"

Par le nouveau code:

Code:
onlogin="window.location='facebook_connect.forum'"


À très bientôt avec d'autres nouveautés sur forumactif
