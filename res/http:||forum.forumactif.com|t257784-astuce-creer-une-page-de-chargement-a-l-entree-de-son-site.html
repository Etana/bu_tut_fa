Astuce : Créer une page de chargement à l'entrée de son site !

 par Vivi le Mar 1 Déc - 9:25

Bonjour,

Faire une page intermediaire affichant un chargement de la page comme ici en cliquant sur Entrer !

C'est très simple et rapide vous allez voir!

  • Tout d'abord créer une page HTML et la mettre en page d'accueil.

    PA > Modules > HTML > Gestion des pages HTML

    • Créer une nouvelle page HTML


      Voulez-vous utiliser le haut et le bas de page de votre forum? Non
      Utiliser cette page en tant que page d'accueil? Oui (en respectant bien de mettre un lien vers le site Forumactif)



    • Insérer le code suivant au champ "Code HTML" sans la partie rouge (commentaires):

      <BODY BGCOLOR=black> <!-- La couleur du fond de la page -->
      <script src="http://astuforum.free.fr/js/snow.js"></script> <!-- Le script décoration Noël 1 -->
      <script src="http://sd-1.archive-host.com/membres/up/28346071248525984/guirlande.js"></script> <!-- Le script décoration Noël 2 -->
      <center><img src="Adresse_image_de_bienvenue"></center><br><br>
      <center><strong><font color="white" face="Arial">Bienvenue sur le Forum de XXX (Nom de votre forum) </font></strong></center>
      <center><br></center>
      <p style="text-align: center;"><a href="Lien_de_deuxième_page_HTML"><img alt="" src="http://www.icone-gif.com/gif/webmaster/entrer/entrer-gif-002.gif" width="144" height="52"></a></p><p style="text-align: center;"><br></p><p style="text-align: center;"><img src="http://www.icone-gif.com/gif/noel/lignes/noel_lignes027.gif"><br></p>
      </p><div style="text-align: right;"><font style="color: White;" size="1">Site hébergé par <a href="http://www.forumactif.com" target="_blank">forumactif.com</a></font></div></BODY>

    • Enregistrer


  • Créer deuxième page HTML où sera le l'image du chargement.

    • Créer une nouvelle page HTML



      Voulez-vous utiliser le haut et le bas de page de votre forum? Non
      Utiliser cette page en tant que page d'accueil? Non



    • Insérer le code suivant au champ "Code HTML" sans la partie rouge (commentaires):

      <BODY BGCOLOR=black> <!-- La couleur du fond de la page -->
      <div style="text-align: center;">
      <img src="Adresse_image_de_bienvenue">
      <br>
      <br>
      <img src="http://www.icone-gif.com/gif/lignes/lignes_filantes/ligne_filante002.gif">
      <br>
      <br>
      <img src="http://www.icone-gif.com/gif/noel/texte/texte-noel-christmas-006.gif">
      <br>
      </div>
      <script src="http://astuforum.free.fr/js/snow.js"></script> <!-- Le script décoration Noël 1 -->
      <script src="http://sd-1.archive-host.com/membres/up/28346071248525984/guirlande.js"></script> <!-- Le script décoration Noël 2 -->
      <meta HTTP-EQUIV="REFRESH" CONTENT="4; URL=Lien_de_la_page_d'accueil_de_votre_forum">
      </BODY>

      * CONTENT="4; --> le nombre en secondes du chargement ! Dans le lien à mettre c'est celui qui va réellement vers le site (forum ou portail)

    • Enregistrer


  • Retournez ensuite à votre première page (la page d'accueil) et mettez un lien vers la deuxième page que vous venez à l'instant de créer!



A vous de jouer maintenant!
