Personnaliser le choix des couleurs dans l'interface de post


Ce tutoriel ne fonctionne que pour les versions où les templates sont modifiables : c'est à dire : phpBB2 et punBB

Attention: Une édition de template signifie qu'à chaque mise à jour de forumactif entrainant une modification sur cette template, vous devrez vous même faire cette modification ( c'est toujours bien expliqué, mais ça reste une charge supplémentaire de travail ).
Les couleurs qui vous sont proposées pour mettre en valeur les posts sur les forums, ne plaisent pas toujours à l'unanimité, et ne sont parfois pas assorties avec vos thèmes de forum : voici comment les modifier :

A savoir :
* Pour ce tutoriel : il faut être l'administrateur fondateur du forum pour avoir accès aux templates.
* Pour trouver des couleurs qui vous plaisent et en connaitre le code : Code-couleur.com
* Les modifications ne s'appliquent qu'à la boite de réponse "normale" : pas à la boite de réponse "rapide".



1 ) Modifier une couleur :

Rendez vous dans votre Panneau d'administration > Affichage > Templates > Poster et messages privés

Modifiez le template : posting_body Page d'envoi de messages

La partie du template qui nous intéresse est celle là :

Code:
<div style="visibility:hidden" class="select" id="color">
      <button style="color:darkred" onclick="bbfontstyle('[color=darkred]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_DARK_RED}</button>

      <button style="color:red" onclick="bbfontstyle('[color=red]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_RED}</button>

      <button style="color:orange" onclick="bbfontstyle('[color=orange]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_ORANGE}</button>

      <button style="color:brown" onclick="bbfontstyle('[color=brown]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_BROWN}</button>

      <button style="color:yellow" onclick="bbfontstyle('[color=yellow]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_YELLOW}</button>

      <button style="color:green" onclick="bbfontstyle('[color=green]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_GREEN}</button>

      <button style="color:olive" onclick="bbfontstyle('[color=olive]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_OLIVE}</button>

      <button style="color:blue" onclick="bbfontstyle('[color=blue]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_BLUE}</button>

      <button style="color:black" onclick="bbfontstyle('[color=black]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_BLACK}</button>

   </div>

Cette division du template, est la partie qui génère votre boite de couleur.

Très exactement, chaque ligne correspond à une couleur :
exemple:
Code:
<button style="color:black" onclick="bbfontstyle('[color=black]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_BLACK}</button>

La ligne de mon exemple est la ligne qui correspond au Noir sur le forum.

En décortiquant et comprenant comment fonctionne cette ligne, vous pourrez parfaitement personnaliser votre boite de couleur On la coupe en trois parties pour mieux la voir :


Il y a trois zones à modifier :

Partie 1)
<button style="color:black"

La partie en ROUGE = La couleur du texte qui sera afficher dans l'interface du post. (la couleur du nom de la couleur sera écrite)
Ici je mettrais #C39300
(Logiquement, j'écris le nom de la couleur, de la même couleur que celle que je veux proposer aux utilisateur : donc je met le même code partie 1 et 2)

Partie 2)
Code:
onclick="bbfontstyle('[color=black]','[/color]');
Ici vous devez modifier le mot black : à la place vous devez mettre le code couleur que vous souhaitez.
(pour plus de clarté, mettez le même que celui dans laquelle vous écrivez le mot dans l'interface de post)
Ici, je veux proposer la couleur "caramel" à mes forumeurs, donc je met le code couleur #C39300

selectWysiwyg(this,'color');return false">{L_COLOR_BLACK}</button>
Ici c'est la partie en ORANGE qu'il faut modifier : il s'agit du nom sous lequel s'affichera votre couleur dans le menu de selection!
Moi je mets : "Couleur des aida"


Voici ma ligne de code après mes modifications :

Code:
<button style="color:#C39300" onclick="bbfontstyle('[color=#C39300]','[/color]');selectWysiwyg(this,'color');return false">Couleur des aida</button>

J'obtiens donc sur mon forum :



Attention : Dans le template le code pour les couleurs apparait deux fois, il faut donc appliquer vos modifications aux deux paragraphes du template qui correspondent aux couleurs.

2) Puis je supprimer une couleur ?


-Bien sur, il suffit simplement de supprimer la ligne qui correspond à la couleur concernée !

Code:
<button style="color:black" onclick="bbfontstyle('[color=black]','[/color]');selectWysiwyg(this,'color');return false">{L_COLOR_BLACK}</button>

Je veux supprimer le noir : alors je n'ai plus qu'à supprimer cette ligne là !


3) Puis-je rajouter une couleur ?

Comme au dessus, Oui vous pouvez !
Il vous suffit de copier une ligne, de la coller en plus des autres, à la suite des autres, et en suite de la personnaliser selon vos désirs !

Note : Lorsque vous rajoutez une couleur : n'oubliez pas le
en fin de ligne.


Une fois vos modifications de templates faites :
Enregistrez votre template



et n'oubliez pas de le publier
