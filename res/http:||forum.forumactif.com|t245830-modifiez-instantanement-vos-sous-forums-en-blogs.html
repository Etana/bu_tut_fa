Modifiez instantanément vos sous-forums en blogs !

 par Vivi le Lun 24 Aoû 2009 - 10:59

Possibilité de transformer les forums de votre forum en blogs


La différence se situe au niveau de la disposition des messages. En effet, les topics s’afficheront sous la forme d’articles que les utilisateurs pourront commenter à leur guise.

Comment ça marche?


Pour pouvoir utiliser cette fonctionnalité, il faut d'abord activer les blogs via PA > Modules > Blogs > Configuration


Petit rappel: Une fois l'option Non est cochée, toutes les configurations de vos blogs repartiront à 0, mais vous ne perdrez jamais le contenu des messages.


    ► PA > Général > Catégories et forums
    ► Choisir le forum à transformer, cliquer "modifier"
    ► Choisissez le type de forum voulu > Blog
    ► Enregistrer
Vous pouvez également créer directement un nouveau forum du type blog.

On peut facilement modifier la taille de cette police via les css personnalisés (PA > Affichage > Couleurs > Feuille de style). Exemple pour chaque version en mettant la même taille que pour les posts des forums :

    ► phpBB2
Code:
#blog_comments .postbody {
    font-size: 12px;
}
    ► phpBB3
Code:
#blog_comments .clearfix {
    font-size : 12px;
}
    ► PunBB
Code:
#blog_comments .entry-content div {
    font-size : 12px;
}
    ► Invision
Code:
#blog_comments .post-entry {
    font-size : 12px;
}
