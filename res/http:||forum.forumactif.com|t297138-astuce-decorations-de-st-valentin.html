Astuce : Décorations de St Valentin

 par Vivi le Mer 9 Fév 2011 - 9:09

Bonjour,

Nous voila bientôt dans la plus belle saison de l'année... La saison des amours ! Et oui, dans 6 jours nous fêterons la St Valentin !
Et pour cette occasion, nous vous avons préparés une série d'animations à mettre sur vos forums !



Pour installer ces codes, deux méthodes :

La première, il vous suffit de insérer le code dans votre panneau d'administration dans la partie de description de site :

Panneau d'Administration ==> Général ==> Forum | Configuration ==> Description du site !

Résultat : L'objet de décoration s'affichera sur toutes les pages de votre forum.




La deuxième, vous pouvez ajouter plusieurs codes à la fois via les balises :

Panneau d'Administration ==> Général ==> Promotion du forum | Les moteurs de recherche ==> Balises supplémentaires.

Résultat : L'objet de décoration s'affichera sur le portail, la page d'accueil, les pages des sous forums et les pages des sujets sur votre forum.
