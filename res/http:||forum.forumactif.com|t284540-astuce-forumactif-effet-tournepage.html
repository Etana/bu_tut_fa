Astuce: Script Effet Tournepage


our ceux que cela intéresse voici un script effet tournepage (testé en phpbb2).
Vous pouvez constater le résultat ici: ICI.

Cet effet permet, grâce à une application Javascript de décoller la partie en haut, à droite de votre forum pour y insérer une image (gif non supporté). Vous pouvez vous en servir afin de promouvoir votre forum (Publicité ou service) que vous proposez. Quand vous cliquez sur cette image, vous êtes redirigé vers un endroit que vous déterminerez.


Mode d'emploi:

1) Réglage du Script:

- Entrer dans votre navigateur l'adresse suivante: http://perso.numericable.fr/antonywar/tournepages.js afin de récupérer le code source du Javascript. Récupérez ce code et ouvrez "Bloc note" de votre pc (Démarrer >> Tous les programmes >>Accessoires).

- Afin de le personnaliser, nous allons modifier les variables:

Code:
jaaspeel.ad_url = escape('http://forum.forumactif.com/');

Dans cette ligne, indiquez l'adresse de redirection au moment où les membres vont cliquez sur votre image contenue dans le "tournepage".

Code:
jaaspeel.small_image = escape('http://perso.numericable.fr/antonywar/cornersmall.png');

Mettre l'image qui s'affichera en haut, à droite de votre forum

Code:
jaaspeel.big_image = escape('http://perso.numericable.fr/antonywar/cornerbig.png');

Mettre l'image qui s'affichera quand vous passerez la souris sur l'onglet en haut à droite. (Message publicitaire).

Code:
jaaspeel.big_width = '650';
jaaspeel.big_height = '650';

Vous n'êtes pas obligé de modifier ces variables. Cependant, sachez que vous pouvez modfiier la taille total de l'objet "Tournepage" et de son incidence sur l'affichage de votre forum.

2) Hébergement de votre script:

- Une fois le script modifiez, enregistrez le document avec l'extension .js afin de le convertir.

- Ensuite, allez sur un Hébergeur JS.

3) Mise en place du fichier sur votre forum:

Pour installer ce script coller ce code dans:

Panneau d'Administration >> Général >> Configuration >> Et collez le code dans la description du site et enregistrez.

Code:
<script src="Lien JS obtenu sur l"hébergeur" /></script>

4) Complément:

Si vous souhaitez mettre votre onglet en haut, à gauche:

Rechercher le code suivant:

Code:
document.write('<div id="jcornerBig" style="position:absolute;width:'+ jaaspeel.big_width +'px;height:'+ jaaspeel.big_height +'px;z-index:9999;left:0px;top:0px;">');

et remplacez le par celui-ci:

Code:
document.write('<div id="jcornerBig" style="position:absolute;width:'+ jaaspeel.big_width +'px;height:'+ jaaspeel.big_height +'px;z-index:9999;right:0px;top:0px;">');
