Vider le cache Internet


Définition du cache :

Le cache du navigateur est un dossier dans lequel votre navigateur enregistre les pages Web que vous visitez (ainsi que tous les éléments de ces pages : images, scripts, styles CSS, clips vidéo...) afin d'accélérer l'affichage de ces pages la prochaine fois que vous les visiterez (la lecture des pages sera plus rapide depuis le disque dur de votre ordinateur que depuis le site Web).

Normalement, lors de l'affichage d'une page Web, le navigateur doit vérifier sur le site Web si les fichiers sont plus récents que ceux du cache, afin de déterminer s'il doit les afficher depuis son cache ou les télécharger à nouveau... mais dans les faits, ce n'est pas toujours le cas.

Il convient alors de "purger le cache" afin de forcer le navigateur à télécharger à nouveau les fichiers de la page Web visitée.

Les méthodes de rafraichissement :


    L'actualisation simple d'une page web :

      L'appui sur la touche de votre clavier force votre navigateur à vérifier, sur le site web visité, la présence d'une version de page plus récente que celle affichée - Il se peut que cette vérification échoue, et que la mise à jour ne s'affiche pas.


    L'actualisation complète d'une page Web :

      L'appui simultané sur les touches + force votre navigateur à effacer de son cache les éléments de la page courante, et à les télécharger à nouveau - Cette manipulation suffit dans 99% des cas à assurer l'actualisation de la page avec succès.


    La purge complète du cache des fichiers internet temporaires :

      Cette manipulation est plus radicale : elle consiste à supprimer du disque dur toutes les pages et tous les éléments web enregistrés par le navigateur dans les "Fichiers internet temporaires".

      En fonction du navigateur utilisé, cette opération permettra également de supprimer de votre disque dur toutes traces de surf :

      • Historique de navigation
      • Cookies
      • Mots de passe enregistrés
      • Saisie semi-automatique
      Cette opération sera détaillée, pour chaque navigateur, dans les chapitres suivants :

        ► Purge du cache d'Internet Explorer
        ► Purge du cache de Firefox
        ► Purge du cache de Google Chrome
        ► Purge du cache du Navigateur Orange
        ► Purge du cache d'Opera
        ► Purge du cache de Safari

Purger le cache de Firefox

 par Lezebre le Lun 7 Jan 2008 - 18:18

Vider le cache Internet ( Firefox )

( les images proviennent de la version 15 de Firefox )


  1. Ouvrir Firefox sur la page d'accueil.

  2. Cliquer sur le menu "Firefox -> Outils >> Supprimer l'historique récent...". (Raccourci : Ctrl + Maj + Suppr)



  3. Cliquer sur "Détails", cocher "Cache" et cliquer sur le bouton "Effacer maintenant".



    Pour effacer également les cookies, il suffira de cocher la case "Cookies" avant de cliquer sur le bouton "Effacer"

Purger le cache d'Internet Explorer

 par Lezebre le Mer 9 Jan 2008 - 9:52

Vider le cache Internet ( Internet Explorer )

( les images proviennent de la version 9 de Internet Explorer )


  1. Ouvrir Internet Explorer.

  2. Ouvrir les Outils -> Options Internet.

  3. Cliquer sur "Supprimer...".



  4. Sélectionner "Fichiers Internet temporaires" et cliquer ensuite sur "Supprimer".



    Pour effacer les Cookies du navigateur, cliquer sur "Supprimer les cookies...". Les paramètres de connexion automatique seront effacés.

Purger le cache d'Opera

 par Lezebre le Mer 9 Jan 2008 - 9:53

Vider le cache Internet ( Opera )

( les images proviennent de la version 12 de Opera )


  1. Ouvrir Opera 12.

  2. Ouvrir le menu Opéra -> Réglages -> Supprimer les données privées...".



  3. Cliquer sur "Options détaillées", cocher la case "Effacer tout le cache" et valider en cliquant sur le bouton "Supprimer".



    Pour effacer également les cookies, il suffira de cocher la case "Supprimer tous les Cookies" avant de cliquer sur le bouton "Effacer"
    Le bouton "Gérer les cookies" permet de n'effacer qu'un ou plusieurs cookies sans supprimer tous les autres (pour les utilisateurs avancés).

Purger le cache de Google Chrome

 par Ea le Mer 5 Aoû 2009 - 14:18

Vider le cache Internet ( Google Chrome )

( les images proviennent de la version 3 de Google Chrome )


  1. Ouvrir Google Chrome sur la page d'accueil.

  2. Cliquer sur "l'icône outils -> Effacer les données de navigation...".



  3. Cocher la case "Vider le cache", régler "Effacer les données de cette période" à "Tout", cliquer sur le bouton "Effacer les données de navigation" et ensuite patienter tant qu'il est mis "Opération en cours...".



    Pour supprimer les cookies, il suffira de cocher la case "Supprimer les cookies" avant de cliquer sur le bouton "Effacer les données de navigation"

Purger le cache du Navigateur Orange

 par Ea le Mer 5 Aoû 2009 - 14:20

Vider le cache Internet ( Navigateur Orange )

( les images proviennent de la version 2 du Navigateur Orange )


  1. Ouvrir le Navigateur Orange sur la page d'accueil.

  2. Cliquer sur le menu "Préférences > Effacer mes traces de navigation...".



  3. Cliquer sur le bouton "Effacer les fichiers temporaires" et ensuite cliquer sur le bouton "Oui" à l'invité avec la question "Voulez-vous vraiment effacer les fichiers temporaires?".



    Pour supprimer les cookies, il suffira de cliquer sur le bouton "Effacer les cookies" et de répondre "Oui" à l'invité demandant "Voulez-vous vraiment effacer les cookies?"
